OriGami Tutorial


1. Creating a Game

  • First Step: Creating an empty Game

    • To create an empty game use the plus sign in the top right corner



  • Second Step: Choosing title, description, time and difficulty



  • Third Step: Choosing an Activity Type


    • Find Destination is a game in which the goal is to move to a given point on the map and optionally solve some kind of task
    • Follow Route is the game consisting of multiple points to reach with the same kind of optional tasks



  • Fourth Step: Adding one (Find Destination) or multiple (Follow Route) points to the game



  • Fifth Step: Adding tasks to the points


    • A Question and Answer task consists of a question and four answers.
    • The first answer has to be the correct one.
    • Each question answer can be expanded by a picture.
    • Multiple Question and Answer games can be assigned to the same point.



    • A GeoReference task consists of a marker which shows the location of where the uploaded photo was taken.

2. Managing Games


  • Deleting Games:

    • To delete a game use the delete button



  • Editing Games

    • To edit existing Games use the edit button



  • Saving a Game locally

    • To save a Game locally use the download button



How to Play OriGami

  • First Step: Choosing a Game

    • To choose a Game select it from the gamelist


  • Second Step: Start Playing

    • Follow the instructions to play the Game
